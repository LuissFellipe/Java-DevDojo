# ☕ Maratona Java — Virado no Jiraya

> Laboratório pessoal de estudos baseado na **Maratona Java** ministrada por [William Suane](https://www.youtube.com/@DevDojoBrasil) no canal DevDojo.

O objetivo é consolidar os fundamentos da linguagem Java — desde tipos primitivos até tópicos avançados de orientação a objetos e coleções.

---

```
maratona-java/

```

---

## 🚀 Como Executar

> Requer **JDK instalado**. Testado em Linux (CachyOS / Arch).

```bash
# 1. Clonar o repositório
git clone https://github.com/LuissFellipe/Java-DevDojo.git

# 2. Entrar na pasta do exercício (exemplo)
cd "maratona-java/src/academy/devdojo/maratonajava/javacore/Aintroducaoclasses/teste"

# 3. Compilar e executar
javac ProfessorTest01.java && java ProfessorTest01
```
